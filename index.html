LMS Live Chat

Project Background

The live chat feature will give students a line of communication to anyone in the class, directly from a class's blackboard page. Send a message directly to other members of the class, or over a channel to reach several people at once. The professor will be able to create channels for relevant discussion topics, to keep information organized. The professor will also be able to promote students or a teacher's assistant to admin in order to monitor the chat room and delete any messages. This feature will make collaboration easier and allow for greater student success.

Project Description

The Learning Management Solutions(LMS) we chose to focus on is Blackboard, which is very popular amongst collegeâ€™s. While analyzing the website for problems, we noticed that there is no default functionality that facilitates communication between members of the class. This is a big issue because if you need to communicate with other members of the class you need to set up some predetermined system of communication. This can get complicated because you need to find a common application you both use from the variety of communication applications such as Discord, Slack, SMS, WhatsApp, Snapchat, or many others. With this project we hope to take out this unnecessary step and give the students and teachers a common direct line of communication right in Blackboard.
